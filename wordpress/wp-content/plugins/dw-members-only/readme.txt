== Description ==

Do you have a private blog that you only want your friends and family to read? Then this plugin is for you. It will redirect any users who aren't logged in to the site to the login form. They will know no difference as there is no message displayed. It is just a simple re-direction script for non-logged in users.

== Installation ==

There are two ways to install this plugin.

= Method 1 - WordPress Upload =
1. Go to your add new plugins page and click on the upload link.
2. Find where you have downloaded the file and upload it to your website.
3. Activate the plugin once uploaded.

= Method 2 - FTP Upload =

1. Extract the downloaded file to your desktop.
2. Upload `dw-admin-footer` folder to the `/wp-content/plugins/` directory using an FTP client.
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
